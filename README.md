# Process of Development
1. Make Branch for development
  ```
  git checkout -b DEV-#
  ```
2. Do development
3. Execute the following check
  ```
 - Rubocop
 - slim-lint(if necessary)
 - scss-lint(if necessary)
 - Rspec(if necessary)
   ```
4. Execute "git push"
5. Make PR as below
  - Template of PR

  ```
  close #0

  # Issues

  # Contents of check
  ```
